# update-python3
 
# ID
Sebuah program untuk update python dari versi 3.6 ke versi 3.7 untuk orang pemalas seperti saya :V hehe

Cara menggunakan

* apt-get install git
* git clone https://github.com/akasakaid/update-python3.git
* cd update-python3
* sh up3.sh
* jika ada pertanyaa, masukkan "yes/y"
* done
